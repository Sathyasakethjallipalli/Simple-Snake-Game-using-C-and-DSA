# Snake Game Using DSA

A simple Snake Game developed using C++ and Data Structures.

## Features
- Snake movement
- Fruit generation
- Score calculation
- Snake growth
- Collision detection
- Console-based gameplay

## Data Structures Used
- Deque
- Pair
- Arrays and coordinates

## Controls
| Key | Action |
|-----|--------|
| W | Move Up |
| S | Move Down |
| A | Move Left |
| D | Move Right |
| X | Exit Game |

## Technologies Used
- C++
- DSA Concepts
- Code::Blocks / VS Code / Dev C++

## How to Run
1. Open `main.cpp` in any C++ compiler.
2. Compile the code.
3. Run the program.
4. Use W, A, S, D keys to play.

## Project Topic
Snake Game using Data Structures and Algorithms.
